# Login - Register System in Laravel
 A Login & Register Sytem with Myprofile and Users List built Using Laravel

<h1>Demo of working URL</h1>

 1.1 See how to set up a locall host <a href="https://www.awordfromnet.com/install-xampp-setup-localhost/" target="_blank">here</a>
 
2. Please Import Job.sql using phpmyadmin to a database of your choice

3. to set up a connection, please modify the .env file and update you database connection settings to point to your database.

4. Bravo, you should be able to see this screen when you access using your project url e.g http://localhost/your-project-url
